# VietNerm Docs

Chào mừng bạn đến với trang tài liệu của **VietNerm**.

- Xem chi tiết dự án tại phần **Tổng quan dự án** trong menu.
- Toàn bộ nội dung Markdown trong `docs/www` sẽ được render thành website bằng MkDocs.

## Cách viết thêm bài mới

1. Tạo file mới trong `docs/www`, ví dụ: `blog-01.md`.
2. Thêm mục vào `nav` trong `mkdocs.yml`.
3. Push lên branch `main` để GitHub Actions tự deploy GitHub Pages.
